# Python在线课程学习管理系统 python1104

## 项目概述


## 技术栈

- 后端：Python 3.8、Django
- 前端：Vue2
- 数据库：MySQL
- 开发环境：PyCharm

## 系统角色

- 管理员
- 用户

## 运行环境

- 服务器：推荐配置Node版本高于14的环境
- 客户端：兼容主流浏览器

## 目录结构

```
project/
├── backend/              # 后端代码目录
│   ├── apps/             # 应用目录
│   ├── manage.py         # 管理脚本
│   └── ...
└── frontend/             # 前端代码目录
    ├── assets/           # 静态资源
    ├── src/              # Vue源码
    ├── package.json      # Node包管理文件
    └── ...
```

## 核心亮点

- 前后端分离：便于团队协作和开发维护
- 丰富的文档：包含Visio图表模板，例如用例图、ER图等
- 开发工具提供：项目所需的全套软件支持
- 培训资料：提供视频教程、同类项目技术讲解和功能实现案例

## 部署步骤

1. 准备符合要求的后端和前端环境
2. 克隆或下载项目代码
3. 按照文档配置数据库和项目依赖
4. 执行数据库迁移和静态资源收集
5. 启动前后端服务
